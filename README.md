# second_brain

A working implementation of the `LLM Wiki` pattern for maintaining shared context between a human operator and an LLM agent.

## Purpose

This repository stores project state, decisions, conventions, and accumulated knowledge in Markdown so future LLM sessions can resume with minimal re-explanation.
