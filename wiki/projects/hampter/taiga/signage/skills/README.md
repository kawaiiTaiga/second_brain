## Rules

- Treat the project as a natural-language-to-formula authoring system, not as a preset picker.
- Prefer low-level channel changes over inventing high-level animation labels.
- When prompt rules change, update the durable knowledge pages in the same pass.
- When runtime semantics change, keep the simulator code and wiki notes synchronized.
