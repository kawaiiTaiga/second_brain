# taiga Skills

Read this page after [../index.md](../index.md). This is the parent-level workflow hub for the TAIGA monorepo.

## Rules

- Keep repo-wide workflow here.
- Keep runtime-specific procedure inside the matching subproject `skills/` folder.
- Before editing, decide whether the task belongs to `taiga-face/`, `signage/`, or both.
- When adding or moving a subproject, update the parent `index.md`, `status.md`, and `map.md` in the same pass.
